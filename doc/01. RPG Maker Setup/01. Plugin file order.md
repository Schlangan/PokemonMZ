# Plugin file order

## 1. PokemonMZ_Core_Configuration
This plugin contains several parameters that you need to configure.

- Max Pokemon level : Max level for Pokemon. They won't be able to gain any more levels.
- Pokemon mechanics : Generation mechanics. For this demo, leave it to 1.
- Player actor Id : ID in the Database Actors for the main character.
- Badges Item IDS : ID for badges. They are used for badges icon in the Player Status Window.
- xxx Animation : Select the animation for the corresponding battle effect
- xxx Sound : Select the SE file for the corresponding sound effect
- xxx ME : select the ME file for the corresponding musical effect
- xxx BGM : select the BGM file for the corresponding bgm sequence

## 2. PokemonMZ_Core_Objects
