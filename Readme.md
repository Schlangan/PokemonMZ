The PokemonMZ plugin is a base for creating Pokemon games with RPG Maker MZ. This first beta version has been developed by mimicking the game mechanics of pokemon Red/Blue/Yellow, up to the Viridian Forest, with pokemon max level limited to 15. Many mechanics have yet to be implemented, but you'll find everything that was accessible up to that point of the original game story.

To use the plugin, push the js/plugin and img/ files inside your project folder. If you want to get the preconfigured database for the current state of this plugin, take the files inside predefinedData/demo/data and paste the contents in the data folder of your project.

Please read the documentation inside the doc folder to configure the plugin.
